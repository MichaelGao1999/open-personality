# 前端 UI 重设计计划 — "Dopamine Personality"

## 设计方向

**核心概念**：多巴胺配色 — 年轻、高饱和、鲜艳、快乐。每个人格维度对应一个高饱和度色彩，整体视觉充满活力和能量。

**美学关键词**：高饱和、鲜艳、年轻、活力、圆润、渐变、快乐

**配色方案（多巴胺五维度）**：

| 维度 | 旧色 | 新色（高饱和） | 色值 |
|------|------|---------------|------|
| 开放性 (O) | #9B7EBD 暗紫 | 亮紫 | #7B2FF7 |
| 尽责性 (C) | #5C6B73 灰 | 电光蓝 | #00B4D8 |
| 外向性 (E) | #E87D4E 暗橙 | 阳光黄 | #FFD60A |
| 宜人性 (A) | #6BA368 暗绿 | 青柠绿 | #56CFE1 |
| 神经质 (N) | #4A6FA5 暗蓝 | 热力粉 | #FF006E |
| **主色** | — | 渐变混合 | 粉→紫→蓝 |
| **背景** | #FFFFFF | 浅灰白 | #FAFAFA |
| **卡片** | 白 | 纯白 | #FFFFFF |
| **文本** | #1E293B | 深灰 | #1A1A2E |

**与当前实现的差异**：
| 项 | 当前 | 重设计 |
|----|------|--------|
| 配色 | 低饱和暗色系 | 高饱和多巴胺色系 |
| 背景 | 纯白 | 浅灰白 + 彩色渐变装饰 |
| 卡片 | 白色 + 阴影 | 白色 + 彩色边框 + 大圆角 |
| 按钮 | 单色 | 渐变色 + hover 变化 |
| 进度条 | 单色 | 五色渐变 |
| 动画 | 基础 hover | 弹性动画 + 彩色脉冲 |
| 圆角 | 8-12px | 16-24px（更圆润） |
| 字体 | Playfair Display | 保留，增加字重 |

## 关键视觉元素

### 渐变按钮
```css
.dopamine-btn {
  background: linear-gradient(135deg, #FF006E, #7B2FF7);
  border: none;
  border-radius: 24px;
  padding: 14px 36px;
  color: white;
  font-weight: 600;
  cursor: pointer;
  transition: transform 0.2s cubic-bezier(0.34, 1.56, 0.64, 1);
}
.dopamine-btn:hover {
  transform: scale(1.05);
}
```

### 五色渐变进度条
```css
.progress-fill {
  background: linear-gradient(90deg, #7B2FF7, #00B4D8, #FFD60A, #56CFE1, #FF006E);
  border-radius: 8px;
  height: 10px;
}
```

### 彩色卡片边框
```css
.dopamine-card {
  background: white;
  border-radius: 20px;
  border: 3px solid transparent;
  box-shadow: 0 4px 20px rgba(0,0,0,0.08);
}
```

### 弹性动画
```css
@keyframes bounceIn {
  0% { transform: scale(0.9); opacity: 0; }
  60% { transform: scale(1.02); }
  100% { transform: scale(1); opacity: 1; }
}
```

## 执行顺序

1. **Phase 1**：创建 `global.css` + 重构 `index.html` CSS 变量
2. **Phase 2**：重构 `App.vue`（过渡 + 背景装饰）
3. **Phase 3**：重构 `HomePage.vue`（首页）
4. **Phase 4**：重构 `QuestionnairePage.vue`（答题页）
5. **Phase 5**：重构 `ReportPage.vue` + `ResultCard.vue`（报告页）
6. **Phase 6**：更新子组件（LanguageSwitch, ShareCodeInput, ShareLink, EasterEggBanner, RadarChart）
7. **Phase 7**：验证 `npm run build` + 视觉检查

---

## 验证方式

1. `npm run build` 构建通过
2. `npm run dev` 手动检查：
   - 首页：渐变标题 + 彩色模式卡片 + 弹性按钮
   - 答题页：五色渐变进度条 + 维度色选项按钮
   - 报告页：渐变标题 + 彩色结果卡片 + 雷达图
   - 语言切换正常
   - 页面过渡弹性动画
3. 五维度色全部正确应用
4. 字体层级正确（Playfair Display / Inter / JetBrains Mono）
